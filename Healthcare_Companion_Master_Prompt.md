# Healthcare Companion – Master Prompt for VS Code AI / Antigravity

## Project Overview
Build a modern, responsive full-stack web application called **Healthcare Companion**.
This is a hackathon prototype that centralizes essential healthcare services into one dashboard.

### Important Requirements
- required login or signup.
- Store all user data using public Storage (or a simple database if configured).
- Responsive for desktop, tablet, and mobile.
- Professional healthcare theme (blue, white, green).
- Clean, reusable, modular code.
- Use React + Vite (preferred) with Node.js/Express (or Flask if preferred).
